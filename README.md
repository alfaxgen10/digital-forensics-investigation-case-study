# Digital Forensics Investigation & Incident Analysis

## Overview
This repository showcases a full digital forensic investigation project conducted as part of academic coursework at Asia Pacific University (APU).  
The project simulates real-world DFIR workflows including evidence acquisition, analysis, malware identification, and formal reporting.

## Workflow
- Evidence acquisition: Live system, RAM, and HDD imaging
- Tool-based analysis: Wireshark, FTK Imager, Autopsy, Volatility, VirusTotal
- Malware identification and artifact analysis
- Data reconstruction and correlation across multiple sources
- Chain of custody documentation
- Formal forensic reporting

## Tools Used
- **Wireshark** – Network traffic capture and analysis
- **FTK Imager** – Disk imaging and evidence preservation
- **Autopsy** – File system and timeline analysis
- **Volatility** – Memory forensics
- **VirusTotal** – Malware verification

This was a **team academic project** completed as part of the Advanced Forensic Methods course.  
It demonstrates applied forensic methodologies aligned with real-world incident response and DFIR practices.
